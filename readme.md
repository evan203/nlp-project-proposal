## Proposal Guidelines

[Proposed project ideas slides](https://docs.google.com/presentation/d/1h3bik7cSHcQaYs2LAS1T02HA0SBL_Yc1t-CrmWMdfAQ/edit?usp=sharing)

For HW3, we expect you to submit some exploratory analysis and a concrete direction that your group will pursue. This progress will be evaluated through four components: Slides (5%), Report (5%), Code (5%), and Presentation (5%).

- The in-person components are the Slides (5%) and Presentation (5%). Students will be divided into small groups, each facilitated by one of the TAs or the professor, and you will present your progress to your assigned group. Each presentation should be approximately 7 minutes long (5 minutes for the presentation and 2 minutes for Q&A). The presentation date is listed as 3/6 on the course website. Any schedule changes will be communicated via an announcement.
- The Report (5%) and Code (5%) components will be submitted on Canvas. The report should cover things like introduction, overview of the exploratory analysis you have conducted, a brief survey of related work, any accompanying graphs or tables, EDA, and a proposed timeline. The goal is to demonstrate that you have a clear idea of what you plan to do and how you intend to achieve it.
- This report can be upto 5 pages. References are not included in the page count. The report must use the official ACL style templates, which are available here Links to an external site. (Latex and Word). Please follow the paper formatting guidelines general to ACL conferences available here Links to an external site.. The proposal report should include the following information:
  - Project title and list of group members.
  - Overview of project idea.
  - A short literature survey of 4 or more relevant papers.
  - Description of methods.
  - Description of data sets to use for your experiments, the experimental settings, and the experiments to be conducted. This should be approximately one page long.
  - Exploratory analysis on the data.
  - Plan of activities, including what you plan to complete by the final report and how you plan to divide up the work among the group members. This should be approximately one page long.
- It is OK to submit appendices that include supplementary information such as hyperparameter settings or additional output examples, although there is no guarantee that the grader will read them. Submissions that exceed the page count will be penalized one third grade (33%) for each page over.

- Please note that the project ideas presented were intended as overviews. It is up to you to design experiments that meet the project expectations.
