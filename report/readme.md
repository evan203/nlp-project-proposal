# Report

Right now this is just boilerplate.

## Technical Details

`main.typ` uses [tracl, a ACL style template](https://github.com/coli-saar/tracl) for [typst](https://typst.app/docs/reference/syntax/).

For editing, check out [this vscode extension](https://marketplace.visualstudio.com/items?itemName=myriad-dreamin.tinymist).

You can build the pdf using `typst compile main.typ`.
