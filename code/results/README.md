# Results

Generated and report-facing outputs live here.

| Directory | Contents |
|---|---|
| `benchmark/` | JailbreakBench ASR, harmless compliance, and perplexity summaries/plots for base, DIM, and ActSVD models. |
| `method_overlap/` | DIM-vs-ActSVD MSO plots and cross-model DIM cosine plots. |
| `safety_utility_overlap/` | Direct activation-space safety-vs-utility MSO metrics and plots. |
| `geometry_repind/` | Geometry-paper-style RepInd cosine-profile comparisons and plots. |

Large model checkpoints do not belong here. Keep them in `code/llm_weights/` or method-specific ignored output directories.
