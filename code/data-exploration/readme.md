# Exploratory Data Analysis

## Load Datasets
This file will load the four datasets into ./data. When you run it, it will ask for a huggingface token. You can create huggingface tokens [at this link](https://huggingface.co/settings/tokens). The **Repositories --> Read Read access to contents of all public gated repos you can access** setting must be checked for the token to function, and your account must have requested access to the datasets.
