# Exploratory Data Analysis

## Load Datasets
This file will load the two datasets into ./data, as well as creating a custom Torch Dataset class to load them. When you run it, it will ask for a huggingface token. You can create huggingface tokens [at this link](https://huggingface.co/settings/tokens). The **Repositories --> Read Read access to contents of all public gated repos you can access** setting must be checked for the token to function.S
